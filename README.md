# 3dsdl

## Description

A basic 3D engine experiment using SDL2 for rendering and Dear ImGui for HUD elements, gorilla-style.

## Dependencies, for now

- pkg-config
- libsdl2-dev

## Build & Run

- `make` to build
- `./bin/3dsdl` to run

Windows builds: Also straightforward if you have mingw64 and pkg-config pointing to the other libs.

## Controls

- WASD: movement
- Left-shift (hold): sprint
- Space: jump
- Mouse-wheel scrolling: adjust FOV
