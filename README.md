# 3dsdl

## Description

A basic 3D engine experiment using SDL2 for rendering and Dear ImGui for HUD elements, gorilla-style.

## Dependencies, for now

- pkg-config
- libsdl2-dev

## Build & Run

- `make` to build
- `./bin/3dsdl` to run

Windows builds: Also straightforward if you have [w64devkit](https://github.com/skeeto/w64devkit) or any other [mingw-w64](https://www.mingw-w64.org/) setup.

## Controls

- WASD: movement
- Left-shift (hold): sprint
- Space: jump
